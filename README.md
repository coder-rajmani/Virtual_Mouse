# Virtual Mouse Using Hand Gestures

This project allows you to control the mouse using hand gestures captured via a webcam using computer vision techniques.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## Features
- Move cursor using index finger
- Left click using index finger + thumb
- Right click using thumb + ring finger
- Scroll using index + middle finger movement
- Smooth cursor movement and scroll control

## How to Run
1. Install dependencies
   pip install opencv-python mediapipe pyautogui

2. Run the program
   python virtual_mouse.py
